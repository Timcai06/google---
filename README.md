# 单词翻译助手 Chrome 扩展

一个优雅的Chrome浏览器扩展，为英语学习者提供强大的划词翻译和单词管理功能。

## ✨ 功能特性

### 🎯 核心功能
- **智能划词翻译** - 在任意网页上选中英文单词、词组或句子，自动弹出翻译结果
- **自动记录** - 智能记录所有翻译过的内容，按单词、词组、句子分类存储
- **高亮显示** - 已翻译的单词在网页上自动高亮显示，便于识别和复习
- **使用统计** - 记录每个单词的使用频率和最近使用时间

### 📊 数据管理
- **分类查看** - 主页显示单词、词组、句子、星标单词四个类别的统计
- **智能搜索** - 在记录页面快速搜索特定单词或内容
- **灵活排序** - 支持按使用次数、最近使用时间、字母顺序排序
- **批量操作** - 支持单个删除或清空所有记录

### 🎨 界面设计
- **现代化UI** - 采用毛玻璃效果和渐变背景，视觉体验优雅
- **响应式设计** - 适配不同屏幕尺寸，提供流畅的用户体验
- **动画效果** - 丰富的悬停和过渡动画，提升交互体验

## 🚀 安装方法

### 方法一：从源码安装（开发者模式）
1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，在地址栏输入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含 `manifest.json` 的项目文件夹

### 方法二：打包安装
1. 在扩展管理页面点击"打包扩展程序"
2. 选择项目根目录，点击"打包扩展程序"
3. 使用生成的 `.crx` 文件安装

## 📖 使用指南

### 基本使用
1. **翻译内容**：在任意网页上选中英文单词、词组或句子
2. **查看翻译**：选中内容后会自动弹出翻译结果
3. **访问记录**：点击浏览器工具栏中的扩展图标

### 记录管理
- **主页概览**：查看各类翻译内容的统计数量
- **详细列表**：点击对应卡片进入详细记录页面
- **搜索功能**：使用搜索框快速定位单词
- **排序选项**：按不同标准重新排列记录
- **星标功能**：标记重要单词便于重点复习
- **删除操作**：移除不需要的翻译记录

## 📁 项目结构

```
单词翻译助手/
├── manifest.json          # 扩展配置文件 (Manifest V3)
├── background.js          # 后台服务脚本
├── content.js             # 内容脚本（处理网页交互）
├── content.css            # 内容脚本样式（翻译弹窗等）
├── popup.html             # 扩展弹窗页面
├── popup.js               # 弹窗页面逻辑
├── popup.css              # 弹窗页面样式（现代化UI）
├── 图标说明.txt           # 图标文件使用说明
├── icon16.png             # 工具栏图标 (16x16)
├── icon48.png             # 扩展管理图标 (48x48)
└── icon128.png            # Chrome商店图标 (128x128)
```

## 🔧 技术栈

- **Manifest版本**: V3 (最新Chrome扩展标准)
- **前端技术**: HTML5, CSS3, JavaScript (ES6+)
- **样式特性**: CSS Grid/Flexbox, 毛玻璃效果, CSS渐变
- **存储方式**: Chrome Storage API (本地存储)
- **翻译服务**: MyMemory翻译API (免费)
- **兼容性**: Chrome 88+

## ⚙️ 配置说明

### 权限要求
- `storage` - 本地数据存储
- `activeTab` - 当前标签页访问权限
- `host_permissions` - 所有网站访问权限（用于翻译功能）

### 图标文件
扩展需要三个PNG格式的图标文件：
- `icon16.png` (16×16px) - 浏览器工具栏显示
- `icon48.png` (48×48px) - 扩展管理页面显示
- `icon128.png` (128×128px) - Chrome网上应用店显示

如果缺少图标文件，Chrome会显示默认图标，不影响功能使用。

## 🔒 隐私与安全

- **本地存储** - 所有翻译记录存储在本地，不会上传到服务器
- **网络请求** - 仅在翻译时访问翻译API服务器
- **数据安全** - 不收集任何个人信息或浏览数据
- **开源透明** - 代码完全开源，可自行审查

## 🛠️ 开发与定制

### 环境要求
- Chrome浏览器 (版本88+)
- 基本的Web开发知识 (HTML/CSS/JavaScript)

### 自定义开发
1. 修改 `popup.css` 调整界面样式和颜色主题
2. 编辑 `content.js` 自定义翻译逻辑

- 暖色阳光主题
- 紫色梦幻主题（当前使用）
- 深色科技主题
